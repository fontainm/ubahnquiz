# U-Bahn Quiz

Lerne die U-Bahn-Stationen aus Wien kennen.

https://www.ubahnquiz.at

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
